Swiss Elites Chat
Beta

A natural-language interface to the Elites Suisses knowledge graph, curated by the LESSH project (Université de Lausanne / Universität Bern). Ask questions in English or French about ~58,700 Swiss elites — political, economic, academic, and military figures from the 19th–20th centuries — and the chat returns a SPARQL query against the public endpoint.

The graph currently exposes:

  • Persons (crm:E21) — 58,729 entries with names
  • Birth events (crm:E67) — ~10,000 entries
  • Family relations (crm:P96/P97) — parents and children
  • Organisations & memberships (sdh-slc:C11) — coming soon (currently only 2 entries)

Ontologies: CIDOC CRM + SDHSS (social-life-core, shortcuts). The system uses Retrieval-Augmented Generation (RAG) over example queries and the dataset's VoID description, with SPARQL validation and optional auto-execution.

We value your input! Feedback and bug reports are welcome — file an issue on the LESSH data repository.
